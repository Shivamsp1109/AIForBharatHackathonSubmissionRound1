# Data Setup

This project uses the demand-forecasting-kernels-only dataset (public). Place `train.csv` in this folder.

Expected file:
- `train.csv`

Processed outputs (generated by `scripts/prepare_data.py`):
- `processed/sales_daily.csv`
- `processed/inventory_snapshot.csv`
- `processed/product_catalog.csv`
- `processed/forecast.csv` (optional, created by `scripts/simple_forecast.py` or Colab)

Quick start (local):
```
python scripts/prepare_data.py --input data/train.csv --output data/processed
python scripts/simple_forecast.py --input data/processed/sales_daily.csv --output data/processed/forecast.csv
```
